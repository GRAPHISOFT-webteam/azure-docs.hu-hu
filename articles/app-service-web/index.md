    <h1>Web Apps-dokumentáció</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">Az Azure Web Apps segítségével egy tetszőleges programozási nyelven hozhat létre és üzemeltethet webhelyeket infrastruktúrakezelés nélkül. Automatikus méretezést és magas rendelkezésre állást kínál, támogatja a Windows és a Linux (előzetes verziójú) rendszert is, valamint lehetővé teszi az automatikus telepítéseket a GitHub, Team Services vagy bármely egyéb Git-adattárból. Ismerje meg, hogyan használhatja az Azure Web Apps szolgáltatást a gyors útmutatóink, oktatóanyagaink és példáink segítségével.</p>

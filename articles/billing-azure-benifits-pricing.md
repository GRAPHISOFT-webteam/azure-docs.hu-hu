#哪裡可以找到 Azure 服務的權益和定價資訊？

Microsoft Azure 可協助您快速部署基礎結構和服務，以符合您所有的商務需求。 您可以在全球 19 個 Azure 資料中心區域執行 Windows 和 Linux 應用程式並以企業級 SLA 傳遞。 之優點的 Azure 服務的資訊，請參閱 [Microsoft Azure 優惠詳細資料](https://azure.microsoft.com/support/legal/offer-details/)。 如需定價資訊，請參閱 [Azure 定價頁面](https://azure.microsoft.com/pricing/)。

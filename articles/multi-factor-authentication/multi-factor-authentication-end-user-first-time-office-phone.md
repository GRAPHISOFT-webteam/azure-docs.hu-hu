# 搭配 Azure Multi-Factor Authentication 使用辦公室電話做為連絡方式

如果您要使用辦公室電話做為主要連絡方式，您可以使用這份文件。 它會引導您設定 Multi-Factor Authentication，以利用辦公室電話撥打電話做為連絡方式。

## 使用辦公室電話做為連絡方式

<ol>
<li>從下拉式清單中選取辦公室電話</li>

<center>![安裝程式](./media/multi-factor-authentication-end-user-first-time-office-phone/office.png)</center>


<li>指定您的國家/地區，從下拉式清單中，輸入您的行動電話號碼。</li>
<li>選取您想要使用您的行動電話-簡訊或電話的模式。</li>
<li>按一下 [下一步]。</li>
<li>按一下 [立即驗證] 按鈕。這會撥打您的辦公室電話。請確定您在電話附近。 
<li>按一下 [下一步]</li>
<li>此時，您已設定好連絡方式，而現在就可以開始設定應用程式密碼的非瀏覽器應用程式，例如 Outlook 2010 或更舊版本。如果您未使用這些應用程式，請按一下 [**完成**]。否則，請繼續下一個步驟。
<li>如果您正在使用這些應用程式，則複製提供的應用程式密碼。</li>

<li>貼上到非瀏覽器應用程式複製到剪貼簿的密碼。如需個別的應用程式，例如 Outlook 和 Lync 的步驟，請參閱如何將您的電子郵件中的密碼變更為應用程式密碼，以及如何變更您的應用程式密碼的應用程式中的密碼。</li>
<li>按一下 [完成]。</li>

# DocumentDB SQL 功能提要 PDF 檔

 **DocumentDB SQL 查詢小祕技** 協助您快速撰寫 DocumentDB 資料的查詢中輕鬆列印 PDF 顯示常用查詢、 關鍵字、 內建函數和運算子參考工作表。 

DocumentDB 支援關聯式資料、 階層式和空間查詢 JSON 文件使用 [SQL](documentdb-sql-query.md) 而未指定結構描述或次要索引。 除了標準的 ANSI-SQL 關鍵字和運算子，DocumentDB 也支援 JavaScript 使用者定義函數 (UDF)、JavaScript 運算子和許多的內建函數。

## 下載 DocumentDB SQL 查詢功能提要

下載查詢 SQL 查詢功能提要做為快速參考使用，讓您快速撰寫查詢。 SQL 功能提要 PDF 檔顯示常見的資料擷取查詢，兩則範例取自 JSON 文件。 為了方便使用，您可以將 SQL 查詢功能提要單面列印 在信紙大小 (8.5 x 11 英吋) 的紙張上。

> [!IMPORTANT] 
> Azure az erőforrások létrehozására és kezelésére két különböző üzembe helyezési modellel rendelkezik: [Resource Manager és klasszikus](../../../resource-manager-deployment-model.md). A Resource Manager modellt Virtuálisgép-bővítmények használatával kapcsolatos információkért lásd: [Itt](../extensions-features.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).

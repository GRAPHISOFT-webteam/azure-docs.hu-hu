# 應該選擇雲端服務嗎？

Azure 對於執行的應用程式提供不同的裝載模型。 每個模型都提供不同的服務集，因此請根據您要執行的工作選擇服務集。 雲端服務是您所考慮的選項嗎？

| 計算選項| 對象|
| ------------------ | --------   |
| [應用程式服務]| 可擴充 Web 應用程式、行動應用程式、API 應用程式，以及任何裝置的邏輯應用程式|
| [雲端服務]| 具備更高的作業系統控制能力的高度可用、可擴充的多層式架構雲端應用程式|
| [虛擬機器]| 具備完整作業系統控制能力的自訂 Windows 和 Linux VM|

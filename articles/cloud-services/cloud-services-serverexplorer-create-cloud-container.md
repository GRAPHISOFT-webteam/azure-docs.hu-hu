# 如何使用 Visual Studio 來建立雲端服務容器

若要檢視 **伺服器總管** Visual Studio 中，在功能表列上選擇 [ **檢視**, ，**伺服器總管**。 

1.  展開 **Azure** 節點

2.  如果使用者尚未登入，系統會自動顯示登入對話方塊。 請依照指示登入 Azure 訂用帳戶

3.  以滑鼠右鍵按一下 **定域機組服務** 節點，選擇 [ **建立雲端服務...**

4.  A **Create Cloud Service** 對話方塊就會出現。 填寫 **訂閱**, ，**名稱**, ，**區域或同質群組** 欄位，然後按一下 [ **建立** ] 按鈕以建立空白的雲端服務容器

## 後續步驟

您已經有雲端服務容器，您可以 [建立及部署](cloud-services-how-to-create-deploy.md) 雲端服務。

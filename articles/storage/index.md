#<a name="storage-documentation"></a>Storage-dokumentáció

Az Azure Storage az ügyfelek igényeihez alkalmazkodó, modern, a tartósságra, rendelkezésre állásra és méretezhetőségre tervezett alkalmazásokhoz legmegfelelőbb felhőalapú tárolómegoldás.  Kiválóan méretezhető, rugalmas, globálisan elérhető, és adatforgalmon alapuló automatikus terheléselosztással is bír.  Az Azure Storage a következő négy szolgáltatást biztosítja az alkalmazások szükségleteihez: Blob Storage, Table Storage, Queue Storage és File storage.

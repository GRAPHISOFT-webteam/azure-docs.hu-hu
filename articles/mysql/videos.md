# <a name="azure-database-for-mysql-videos"></a>Azure-adatbázis MySQL-videók

[Első lépések az új Azure-adatbázissal MySQL-szolgáltatás](https://channel9.msdn.com/events/Build/2017/B8045)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8045/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

Ez a videó az lehet, hogy 2017 a Microsoft //Build konferencia, további tudnivalók a Microsoft által felügyelt MySQL ajánlat az Azure-ban. A videó bemutatja, hogyan keresztül stratégiát a Microsoft nyílt forráskódú adatbázis rendszerek támogatásához az Azure-ban. A videó ismerteti, mit jelent, ha Ön fejlesztőként továbbfejlesztésével és MySQL az Azure-ban használó alkalmazások központi telepítése. Ez a videó a szolgáltatás architektúrájának áttekintése látható, és bemutatja a MySQL az Azure-adatbázis integrálva van a más Azure-szolgáltatásokon, például webes alkalmazásokat.

# 在 Azure Machine Learning Studio 中管理實驗逐一查看

開發預測分析模型是一種逐一查看過程 - 您修改實驗的各種函數及參數，結果會不斷收斂，直到您對已訓練的有效模型感到滿意為止。 此程序的關鍵是追蹤實驗參數和組態的各種逐一查看。

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

您可以隨時檢閱先前的實驗執行，以挑戰、重新瀏覽及最終確認或調整先前的假設。 當您執行實驗時，Machine Learning Studio 會保留執行歷程記錄，包括資料集、模組及連接埠連接和參數。 此歷程記錄也會擷取結果、執行階段資訊，例如開始和停止時間、記錄訊息及執行狀態。 您可以隨時回顧任何執行，以檢閱實驗和中繼結果的年表。 您甚至可以使用上一次的實驗執行，啟動到新的階段，在您的路徑上查詢和探索，以建立簡單、複雜或甚至集成模型解決方案。

> [AZURE.NOTE] 當您檢視實驗的上一次執行時，該版本的實驗已鎖定且無法加以編輯。 但是，您可以按一下 [儲存一份 **存** 並提供新名稱的副本。 Machine Learning Studio 會開啟新複本，然後您可以編輯及執行。 實驗的此複本位於 **實驗** 含有所有其他實驗的清單。

## 檢視上一次執行

當您開啟至少一次執行時，可以按一下檢視先前執行的實驗 **先前執行** 屬性] 窗格中。

例如，假設您建立實驗並且在下列時間執行其版本：11:23、11:42 及 11:55。 如果您開啟上次執行實驗 (11:55)，然後按一下 [ **執行**, ，開啟您在 11:42 執行的版本。

## 檢視執行歷程記錄

您可以按一下 [檢視所有先前的實驗執行 **檢視執行歷程記錄** 開啟的實驗中。

例如，假設您建立實驗，其具有 [線性迴歸][linear-regression] 模組，且您想要觀察的變更值的效果 **學習速度** 在實驗結果。 您針對此參數以不同值執行實驗多次，如下所示：

| 學習速度值 | 執行開始時間 |
| ------------------- | -------------- |
| 0.1 | 2014/9/11 下午 4:18:58
| 0.2 | 2014/9/11 下午 4:24:33
| 0.4 | 2014/9/11 下午 4:28:36
| 0.5 | 2014/9/11 下午 4:33:31

如果您按一下 **檢視執行歷程記錄**, ，您會看到一份所有執行 ︰

![範例執行歷程記錄][runhistory]

按一下任何執行以檢視您執行該實驗之時間的實驗快照。 保留組態、參數值、註解及結果，給予您該實驗執行的完整記錄。

> [AZURE.TIP] 若要記載您的實驗逐一查看，您可以修改標題每次您執行它，您可以更新 **摘要** 屬性中實驗的窗格中，而且您可以新增或更新註解，在個別的模組，以記錄您的變更。 標題、摘要及模組註解會在每一次執行實驗時儲存。

清單中的實驗 **實驗** Machine Learning Studio 中的索引標籤一律會顯示最新版本的實驗。 如果您開啟實驗的上一次執行 (使用 **執行** 或 **檢視執行歷程記錄**)，您可以返回草稿版本，即可 **檢視執行歷程記錄** ，然後選取 [反覆項目具有 **狀態** 的 **編輯**。

## 逐一查看上一次執行

當您按一下 **Prior Run** 或 **檢視執行歷程記錄** 和開啟先前執行時，您可以在唯讀模式中檢視完成的實驗。

如果您想要開始設定上一次執行的方式開始實驗反覆項目，您可以藉由開啟執行按 **存**。 這會建立新的實驗，具有新的標題、空白的執行歷程記錄及上一次執行的所有元件和參數值。 這個新的實驗會列在 **實驗** ] 索引標籤，在 Machine Learning Studio 首頁上，而且您可以修改並執行它時，起始新的執行歷程記錄實驗的這個反覆項目。 

例如，假設您有上一個章節顯示的實驗執行歷程記錄。 您想要觀察當您設定時，會發生什麼事 **學習速度** 參數設為 0.4，並且嘗試不同值的 **訓練 epoch 數目** 參數。


1. 按一下 [ **檢視執行歷程記錄** 並開啟您在下午 4:28:36 （這在您設定參數值設為 0.4） 執行的實驗的反覆項目。

2. 按一下 [ **存**。

3. 輸入新的標題，然後按一下 [ **確定** 核取記號。 實驗的新複本隨即建立。

4. 修改 **訓練 epoch 數目** 參數。

5. 按一下 [ **執行**。

您現在可以繼續修改及執行此實驗版本，建立新的執行歷程記錄以記錄您的工作。

# <a name="datamarket-is-now-deprecated"></a>DataMarket most már elavult.

A [Microsoft DataMarket](https://datamarket.azure.com/datasets) most már elavult, és így a Machine Learning API-k nem tárolt is mára elavult.

Most ezeket a képességeket (és további) az található a [kognitív szolgáltatások API-k](https://www.microsoft.com/cognitive-services).

## <a name="text-analytics-api"></a>Szövegelemzések API

Előző DataMarket szöveg Analytics API ügyfelek kell indíthatja a [kognitív szolgáltatások szöveg Analytics API](https://www.microsoft.com/cognitive-services/text-analytics-api).
A kognitív szolgáltatások szöveg Analytics API támogatja az DataMarket régebbi volt, valamint további lehetőségeket nyelvi felderítését és a témakör észlelési forgatókönyvek.

#支援的瀏覽器和裝置
Microsoft Azure 入口網站可在所有最新的桌上型電腦和平板電腦裝置與瀏覽器上執行。

### 支援的裝置
入口網站可在最新的 PC、Mac 和平板電腦上順利執行。 目前尚不支援行動電話。

### 支援的瀏覽器
可支援下列最新版的瀏覽器：

- Edge (最新版本)
- Internet Explorer 11
- Safari (最新版本，僅限 Mac)
- Chrome (最新版本)
- Firefox (最新版本)

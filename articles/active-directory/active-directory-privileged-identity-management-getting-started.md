# 開始使用 Azure 特殊權限身分識別管理

## 啟動特殊權限身分識別管理擴充功能

1.  在 Azure Active Directory 中建立目錄。
2.  將使用者加入 Azure Active Directory。
3.  在目錄中建立幾個全域管理員。
4.  登入 <http://portal.azure.com> 與全域系統管理員帳戶。
5.  按一下 [ **username** 左上角的 [入口網站。
6.  選取 **目錄** 您剛才所建立。
7.  按一下 [ **新增** 在左側導覽中的圖示。
8.  選取 **安全性 + 身分識別** 從 [建立] 功能表。
9.  選取 **Azure AD 特殊權限的身分識別管理** 從 **安全性 + 身分識別** 功能表。
10. 保留 **釘選到開始面板] 核取方塊** 勾選，然後按 **建立** ] 按鈕。 [特殊權限身分識別管理] 儀表板隨即開啟。

# 適合企業使用的 Windows 10：使用裝置的工作方式

Windows 10 讓您可以充分利用 Azure Active Directory。  Windows 10 裝置可以連接到 Azure AD，使用者可以使用 Azure AD 帳戶登入 Windows，或加入其 Azure 識別碼存取商務應用程式和資源。

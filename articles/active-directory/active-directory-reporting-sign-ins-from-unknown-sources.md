# 從不明來源登入
<p>此報表會指出已成功登入您的目錄，同時指派的用戶端 IP 位址被 microsoft 辨識為匿名 proxy IP 位址 (Tor IP) 的使用者。 這些 Proxy 通常由想要隱藏其電腦 IP 位址的使用者使用，而且可能用於惡意意圖。 </p><p> 這份報告的結果會顯示使用者從該位址和 Proxy 的 IP 位址成功登入您目錄的次數。</p>

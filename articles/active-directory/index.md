# <a name="azure-active-directory-documentation"></a>Azure Active Directory-dokumentáció

Az Azure Active Directory (Azure AD) a Microsoft több-bérlős felhőalapú könyvtára és identitáskezelési szolgáltatása.  Az Azure AD identitáskezelési és hozzáférési képességeket biztosít az Azure-ban és a helyszínen futó alkalmazásoknak egyaránt.

## <a name="log-in-to-azure"></a>Jelentkezzen be az Azure-ba

Az Azure CLI-vel létre fogja hozni az alkalmazás Azure-beli üzemeltetéséhez szükséges erőforrásokat. Jelentkezzen be az Azure-előfizetésbe az [az login](/cli/azure/#login) paranccsal, és kövesse a képernyőn látható utasításokat.

```azurecli-interactive
az login
```

您可以使用 Visual Studio 建立 Azure 佇列 **伺服器總管**。

![伺服器總管 Blob][Image1]

1. 在 **檢視** ] 功能表上，選擇 [ **伺服器總管**。
2. 在 [伺服器總管] 中，展開 **Azure** 展開節點，針對您的訂閱， **儲存體** 節點以及您在 Azure 儲存體中指定的儲存體帳戶的節點連接的服務。
3. 選取 **佇列** 節點，然後選擇 **建立佇列** 從內容功能表。
4. 輸入佇列的名稱，然後選擇 **確定**。

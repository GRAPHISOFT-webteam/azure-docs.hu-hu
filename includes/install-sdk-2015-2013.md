## <a name="setupdevenv"></a>設定開發環境

首先，設定開發環境中安裝 [Azure SDK for Visual Studio 2015](http://go.microsoft.com/fwlink/?linkid=518003) 或 [Azure SDK for Visual Studio 2013](http://go.microsoft.com/fwlink/?LinkID=324322)。

如果您尚未安裝 Visual Studio (請使用 Visual Studio 2015 的連結)，Visual Studio 將隨著 SDK 一起安裝。

>[AZURE.NOTE]視您的電腦上有多少 SDK 相依性而定，安裝 SDK 可能需要很長的時間 (從數分鐘到半小時以上不等)。

#### 在虛擬裝置上建立公用端點

1. 登入管理入口網站。

- 按一下 [虛擬機器]****，然後選取想要用來做為虛擬裝置的虛擬機器。

- 按一下 [端點]****。 [端點] 頁面會列出虛擬機器的所有端點。

- 按一下 [ **新增**。 [新增端點] 對話方塊隨即出現。 按一下箭頭以繼續。

- 針對 [名稱]****，為端點輸入下列名稱：**WinRMHttps**。

- 將 [TCP]**** 指定為 [通訊協定]****。

- 針對 [公用連接埠]****，輸入要用於連線的連接埠號碼。

- 針對 [私人連接埠]****，輸入 [5986]****。

- 按一下核取記號以建立端點。

建立端點之後，您可以檢視其詳細資料，以確認公用虛擬 IP (VIP) 位址。 請記下此位址。

#### 在虛擬裝置上建立公用端點

1. 登入管理入口網站。

- 按一下 [ **虛擬機器**, ，然後選取 [使用您的虛擬裝置的虛擬機器。

- 按一下 [ **端點**。 [端點] 頁面會列出虛擬機器的所有端點。

- 按一下 [ **新增**。 [新增端點] 對話方塊隨即出現。 按一下箭頭以繼續。

- 如 **名稱**, ，輸入下列名稱的端點 ︰ **WinRMHttps**。

- 如 **通訊協定**, ，指定 **TCP**。

- 如 **公用連接埠**, ，輸入您要用於連線的連接埠號碼。

- 如 **私用連接埠**, ，型別 **5986**。

- 按一下核取記號以建立端點。

建立端點之後，您可以檢視其詳細資料，以確認公用虛擬 IP (VIP) 位址。 請記下此位址。

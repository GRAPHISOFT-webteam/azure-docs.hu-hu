## <a name="prerequisites"></a>Előfeltételek
Az Eclipse Azure eszköztára szükséges a következő szoftverösszetevőket:

* IDE Holdas Java EE-fejlesztőknek, Mars, vagy később. Letöltheti azt a [Eclipse webhely](http://www.eclipse.org/downloads/).

* A Java fejlesztői készlet (JDK), 1,8 vagy újabb verziója.

* Az operációs rendszer. Az Eclipse Azure eszköztára tesztelték a következő operációs rendszereken:
  
  * Windows 10, Windows 8.1, Windows 8 és Windows 7
  * Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2 és Windows Server 2008
  * [Mac OS X](http://www.apple.com/osx) "Yosemite" verzió és újabb verziók
  * [Ubuntu Linux](http://www.ubuntu.com) 14, 15 és 16

További információkért lásd: a [Eclipse Azure eszköztára](http://marketplace.eclipse.org/content/azure-toolkit-eclipse) az Eclipse piactér lapon.

資料集從 KDD Cup 1999 知識探索和資料採礦工具競賽 (<a href="http://kdd.ics.uci.edu/databases/kddcup99/kddcup99.html">kddcup99.html</a>)。<p> </p>資料集已下載並儲存在 Azure Blob 儲存體 (<a href="https://azuremlsampleexperiments.blob.core.windows.net/datasets/network_intrusion_detection.csv">network_intrusion_detection.csv</a>)，同時包含訓練和測試資料集。 訓練資料集約有 126K 個資料列和 43 個資料行 (包含標籤)；其中 3 個資料行屬於標籤資訊，而 40 個資料行 (包含數值和字串/類別特性) 可用於訓練模型。 測試資料約有 22.5K 個測試範例 (具有與訓練資料相同的 43 個資料行)。

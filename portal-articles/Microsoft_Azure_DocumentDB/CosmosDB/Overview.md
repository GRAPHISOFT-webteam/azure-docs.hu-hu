# <a name="azure-cosmos-db-documentation"></a>Azure Cosmos DB-dokumentáció

Az Azure Cosmos DB egy globálisan elosztott adatbázis-szolgáltatás, amely arra lett kifejlesztve, hogy – átfogó SLA-feltételek mellett – lehetővé tegye az átviteli sebesség és a tárterület rugalmas és független méretezését akár több földrajzi régióra kiterjedően is. A Cosmos DB használatával egy sor népszerű API és programozási modell segítségével fejleszthet dokumentum-, kulcs/érték- és Graph-adatbázisokat. Rövid útmutatóink, oktatóanyagaink és példáink segítségével megismerkedhet a Cosmos DB használatával.
